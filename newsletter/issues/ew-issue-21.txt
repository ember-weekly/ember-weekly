** Ember 1.0 RC8 (Not yet!) Released
------------------------------------------------------------

Not yet official, but since 1.0 final is likely to be out by next weeks issue (yay!), it&#x27;s probably best to give you a heads up. Watch the [official Ember blog](http://emberjs.com/blog/) for further announcements. RC8 has some breaking changes and a few nifty new features have sneaked in. (https://github.com/emberjs/website/blob/rc8-cp-change/source/blog/2013-08-25-ember-1-0-rc8.md)

** Ember 1.0 Documentation Audit
------------------------------------------------------------

A heroic effort by Trek and a load of community members to clean up the API documentation in time for the imminent release of 1.0 (docs not yet live on website). Check out [Trek&#x27;s postmortem](https://gist.github.com/trek/6343879) for more information. Great work everyone! (https://github.com/emberjs/ember.js/pull/3082)


** An introduction to Ember.js

Follow along at the blog and [github repo](https://github.com/jacojoubert/emberui). (https://medium.com/p/dac11785c50f)
