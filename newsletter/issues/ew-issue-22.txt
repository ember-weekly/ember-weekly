ISSUE #22

------------------------------------------------------------
* The Big Release
------------------------------------------------------------


** Ember 1.0 Released
------------------------------------------------------------

Two and a half years in the making but we got there. After a [huge push](https://github.com/emberjs/ember.js/issues/3246) by a large number of (awesome) community members and the (tireless) core team, Ember 1.0 was released on the weekend. A huge congratulations to everyone involved, and thank you for all your hard work. Check out the blog post for more information on 1.0 and see the rest of the newsletter for more updates and releases related to 1.0. (http://emberjs.com/blog/2013/08/31/ember-1-0-released.html)



------------------------------------------------------------
* Other Releases
------------------------------------------------------------


** Ember Data 1.0 Beta 1 &amp; 2 Released
------------------------------------------------------------

Ember Data 1.0 Beta 1 was released [alongside Ember 1.0](http://emberjs.com/blog/2013/08/31/ember-1-0-released.html#toc_ember-data-1-0-beta-1). Beta 2 was hot on it&#x27;s heals with Yahuda Katz tweeting that [more releases should &quot;be coming fast and furious&quot;](https://twitter.com/wycats/status/375447703389339648) (http://emberjs.com/blog/2013/09/04/ember-data-1-0-beta-2-released.html)


** Ember 1.0 RC8 Released
------------------------------------------------------------

Mentioned in last week&#x27;s issue, RC8 was officially released last week with some breaking changes and a few nifty new features which made it in before Ember 1.0. (http://emberjs.com/blog/2013/08/29/ember-1-0-rc8.html)


** Ember Inspector Chrome Plugin
------------------------------------------------------------

This awesome tool for debugging Ember applications has been released to the Chrome Web Store. Big shout out to Teddy Zeenny for all his had work. A truly useful tool every Ember developer should have installed. (https://chrome.google.com/webstore/detail/ember-inspector/bmdblncegkenkacieihfhpjfppoconhi)
