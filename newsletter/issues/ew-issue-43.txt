* Releases
------------------------------------------------------------


** Ember 1.0.0 Beta 6 Available
------------------------------------------------------------

Contains a fix for Ember App Kit based apps to allow resolving serializers and adapters properly. It&#x27;s also worth noting that Ember Data master now uses [ES6 modules and JavaScript build tools](https://github.com/emberjs/data/pull/1623) (no more ruby dependency). (http://emberjs.com/builds/#/beta)


** Ember 1.4 Beta 4 Available
------------------------------------------------------------

Contains various bug fixes. (https://github.com/emberjs/ember.js/blob/v1.4.0-beta.4/CHANGELOG.md)

------------------------------------------------------------


** A Guide to Building Your First Ember.js App
------------------------------------------------------------

A nice introduction to Ember by Balint Erdi, with a walkthrough of how to build a Rock &amp; Roll app. (http://www.toptal.com/javascript/a-step-by-step-guide-to-building-your-first-ember-js-app)


** Our Ember.js Toolchain
------------------------------------------------------------

A look at how NebuLab develop Ember applications with Brunch, tapas-with-ember, Rails, and dyson (to fake an API). (http://nebulab.it/blog/our-ember-js-toolchain)


** Tip: Golang with Ember.js - JSON Response
------------------------------------------------------------

A short summary of building a Ember app with Go and the Martini framework. (http://dukex.svbtle.com/tip-golang-with-emberjs-json-response)


** Execute bad ideas: Ruby on Rails → node.js/ember
------------------------------------------------------------

An amusing story of how Ember was used to improve the UX of this &quot;bad idea&quot;. (https://medium.com/p/bd4ec4e352b2)


** Ruby on Rails MVC Is Not the Same as Ember.js MVC
------------------------------------------------------------

A breakdown of the difference in MVC implementations from Yehuda Katz&#x27;s talk, [A tale of two MVC’s](http://www.youtube.com/watch?v=s1dhXamEAKQ). (http://blog.abuiles.com/blog/2014/01/27/ruby-on-rails-mvc-is-not-the-same-as-ember-dot-js-mvc/)


** Getting Started With Ember.js and Ruby on Rails: Ember Appkit Rails

** Ember.js + Firebase = &lt;3
------------------------------------------------------------

An introduction post to ember-firebase, a new unofficial firebase library with tight integration with Ember bindings ([code](https://github.com/mjijackson/ember-firebase)). (https://github.com/mjijackson/ember-firebase)


** Improved Test Helpers in Ember App Kit
------------------------------------------------------------

Stefan Penner has added some new test helpers into Ember App Kit that should simplify writing tests, especially async and component tests. (https://github.com/stefanpenner/ember-app-kit/pull/452)
