** Ember-Parallel
------------------------------------------------------------

An Ember.js plugin proving integration with Parallel.js. The plugin allows you to run code inside web workers in the browser (i.e. in background threads) and access the results easily within your Ember app. (https://github.com/bengillies/ember-parallel)
